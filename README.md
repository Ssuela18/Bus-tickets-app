# Bus-ticket-app
This is an online platform for booking intercity bus tickets across Albania.
Search your departure and arrival city,time of departure,seats,and pay - all in one place.

## Features
- Search bus routes between Albanian cities
- View schedules and available seats(optional)
- Interactive seat selection
- Passenger details and booking form
- Online payment
